## Ejecucion 

Para la ejecucion del programa se necesitan los siguientes comandos 


- php artisan migrate
- php artisan migrate:fresh
- php artisan db:seed


Antes de ello modificar el archivo .env con su conexion al base de datos 

## modificar el archivo .evn
- usuario
-nombre de la base de datos
-puerto
